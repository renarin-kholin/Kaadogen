# Kaadogen

Figma-inspired ID card batch generator. Import SVG templates, bind elements to JSON data, export as PDF/PNG/SVG.

## Quick Start

```bash
pnpm install
pnpm dev
```

## Usage

1. **Import SVG** - Upload your card template
2. **Load JSON** - Bind data fields to SVG elements
3. **Preview** - Navigate through records
4. **Export** - Generate PDF or ZIP of cards

Built with React + TypeScript + Vite.
